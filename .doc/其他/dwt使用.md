  DWT 在 ARM Cortex-M 中是一个完整的调试/跟踪模块，包含以下功能：

  ┌─────────────────┬─────────────────────────────┬───────────────────────┐
  │      功能       │            说明             │    本文件是否使用     │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ CYCCNT          │ 周期计数器（每CPU时钟+1）   │ 是 — 全部接口都基于它 │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ DWT_COMPn       │ 数据监视点/硬件断点（0~15） │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ Exception Trace │ 异常进出跟踪                │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ PC Sampling     │ 周期性PC采样                │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ FOLDCNT         │ 折叠指令计数器              │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ LSUCNT          │ Load/Store 操作计数器       │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ SLEEPCNT        │ 睡眠周期计数器              │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ EXCCNT          │ 异常开销周期计数器          │ 否                    │
  ├─────────────────┼─────────────────────────────┼───────────────────────┤
  │ CPICNT          │ 每指令周期数计数器          │ 否                    │
  └─────────────────┴─────────────────────────────┴───────────────────────┘
