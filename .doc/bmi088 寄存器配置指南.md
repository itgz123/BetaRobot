# BMI088 寄存器配置指南

## 1. 概述

本文档详细描述 BMI088 六轴惯性测量单元 (IMU) 的所有寄存器地址、功能说明以及配置方法。BMI088 包含两个独立的传感器部分：三轴加速度计和三轴陀螺仪，它们有各自的寄存器映射。

## 2. SPI 通信协议

### 2.1 SPI 读/写操作

BMI088 通过 SPI 接口进行通信，通信协议如下：

- **写操作**：`[寄存器地址] [数据]`
- **读操作**：`[寄存器地址 | 0x80] [dummy 字节] [数据]`

**注意**：加速度计部分在 SPI 读操作时，需要先发送一个 dummy 字节，然后才能读取到有效数据。

### 2.2 片选引脚

- **加速度计**：CSB1 引脚
- **陀螺仪**：CSB2 引脚

#### 3.2.4 寄存器 0x12-0x17：ACC 数据（只读）

| 寄存器 | 名称      | 访问 | 复位值 | 描述             |
| :----- | :-------- | :--- | :----- | :--------------- |
| 0x12   | ACC_X_LSB | RO   | 0x00   | X 轴加速度低字节 |
| 0x13   | ACC_X_MSB | RO   | 0x00   | X 轴加速度高字节 |
| 0x14   | ACC_Y_LSB | RO   | 0x00   | Y 轴加速度低字节 |
| 0x15   | ACC_Y_MSB | RO   | 0x00   | Y 轴加速度高字节 |
| 0x16   | ACC_Z_LSB | RO   | 0x00   | Z 轴加速度低字节 |
| 0x17   | ACC_Z_MSB | RO   | 0x00   | Z 轴加速度高字节 |

**功能**：包含加速度传感器输出的 16 位二进制补码数据。

**数据计算公式**：

```
Accel_X_int16 = ACC_X_MSB * 256 + ACC_X_LSB
Accel_Y_int16 = ACC_Y_MSB * 256 + ACC_Y_LSB
Accel_Z_int16 = ACC_Z_MSB * 256 + ACC_Z_LSB
```

**转换为 mg（毫重力加速度）**：

```
Accel_X_mg = Accel_X_int16 / 32768 * 1000 * 2^(ACC_RANGE+1) * 1.5
```

#### 3.2.7 寄存器 0x22-0x23：温度传感器数据（只读）

| 寄存器 | 名称     | 访问 | 复位值 | 描述           |
| :----- | :------- | :--- | :----- | :------------- |
| 0x22   | TEMP_MSB | RO   | 0x00   | 温度数据高字节 |
| 0x23   | TEMP_LSB | RO   | 0x00   | 温度数据低字节 |

**功能**：包含温度传感器数据，11 位二进制补码格式，分辨率 0.125°C/LSB。

**温度计算公式**：

```
Temp_uint11 = (TEMP_MSB * 8) + (TEMP_LSB / 32)
if Temp_uint11 > 1023:
    Temp_int11 = Temp_uint11 - 2048
else:
    Temp_int11 = Temp_uint11
Temperature = Temp_int11 * 0.125 + 23  (°C)
```

#### 3.2.13 寄存器 0x6D：ACC_SELF_TEST（读/写）

| 位    | 名称        | 访问 | 复位值 | 描述               |
| :---- | :---------- | :--- | :----- | :----------------- |
| [7:5] | -           | -    | -      | 保留               |
| [4]   | -           | RW   | 0x00   | 保留               |
| [3]   | acc_bist_en | RW   | 0x00   | 1=使能自检         |
| [2]   | acc_sign    | RW   | 0x00   | 0=正极性，1=负极性 |
| [1:0] | -           | -    | -      | 保留               |

**功能**：控制加速度计自检功能。

**自检步骤**：

1. 设置量程为±24g：写入 0x03 到 ACC_RANGE (0x41)
2. 设置 ODR=1.6kHz，正常模式：写入 0xAC 到 ACC_CONF (0x40)
3. 等待>2ms
4. 使能正极性自检：写入 0x0D 到 ACC_SELF_TEST (0x6D)
5. 等待>50ms，读取加速度值
6. 使能负极性自检：写入 0x09 到 ACC_SELF_TEST (0x6D)
7. 等待>50ms，读取加速度值
8. 禁用自检：写入 0x00 到 ACC_SELF_TEST (0x6D)
9. 计算正负自检响应的差值，与预期值比较

## 5. 初始化配置流程

### 5.1 加速度计初始化流程

```
1. 上电后等待 1ms
2. 写入 0x04 到 ACC_PWR_CTRL (0x7D) 使能加速度计
3. 等待 450μs
4. 写入 0x03 到 ACC_RANGE (0x41) 设置±24g 量程（自检时需要）
5. 写入 0xAC 到 ACC_CONF (0x40) 设置 ODR=1.6kHz，正常模式
6. 配置中断引脚（如需要）：
   - 写入 0x08 到 INT1_IO_CONF (0x53) 使能 INT1 输出
   - 写入 0x04 到 INT1_INT2_MAP_DATA (0x58) 映射数据就绪到 INT1
```

### 5.2 陀螺仪初始化流程

```
1. 上电后等待 30ms（陀螺仪默认进入正常模式）
2. 写入 0x00 到 GYRO_RANGE (0x0F) 设置±2000°/s 量程
3. 写入 0x01 到 GYRO_BANDWIDTH (0x10) 设置 ODR=2000Hz，带宽 230Hz
4. 写入 0x00 到 GYRO_LPM1 (0x11) 确保正常模式
5. 配置中断（如需要）：
   - 写入 0x80 到 GYRO_INT_CTRL (0x15) 使能数据就绪中断
   - 写入 0x01 到 INT3_INT4_IO_MAP (0x18) 映射数据就绪到 INT3
   - 写入 0x00 到 INT3_INT4_IO_CONF (0x16) 配置推挽、低电平有效
```

### 5.3 SPI 通信示例

**读取加速度计 X 轴数据**：

```
发送：[0x92] [0x55]  // 0x92 = 0x12 | 0x80 (读), 0x55 = dummy
接收：[dummy] [X_LSB] [X_MSB]
```

**写入加速度计配置**：

```
发送：[0x40] [0xAC]  // 0x40 = ACC_CONF, 0xAC = 配置值
```

**读取陀螺仪 X 轴数据**：

```
发送：[0x82]  // 0x82 = 0x02 | 0x80 (读)
接收：[X_LSB] [X_MSB]
```

## 6. 注意事项

1. **SPI 加速度计读操作**：加速度计在 SPI 读操作时需要先发送一个 dummy 字节，接收到的第一个字节也是无效的，需要丢弃。

2. **软复位延时**：
   - 加速度计软复位后等待至少 1ms
   - 陀螺仪软复位后等待至少 30ms

3. **电源模式切换**：
   - 加速度计从挂起模式到正常模式需要 450μs
   - 陀螺仪从挂起/深度挂起模式唤醒需要 30ms

4. **寄存器写入间隔**：在正常模式下，两次写操作之间需要至少 2μs 的间隔；在挂起模式下需要至少 1000μs 的间隔。

5. **数据读取锁定**：读取 LSB 寄存器时会锁定对应的 MSB 寄存器，确保读取的数据属于同一次采样。
