msys2安装方法：  

**`git`,`cmake`,`ninja`,`openocd`使用`msys2`安装方法**  
在`mingw64.exe`运行以下命令
- 先更新`MSYS2`本身
    ```bash
    pacman -Syu
    ```
- 安装`git`
    ```bash
    pacman -S git
    ```
- 安装`cmake`
    ```bash
    pacman -S mingw-w64-x86_64-cmake
    ```
- 安装`ninja`
    ```bash
    pacman -S mingw-w64-x86_64-ninja
    ```
- 安装`openocd`
    ```bash
    pacman -S mingw-w64-x86_64-openocd
    ```

# 框架使用方法

### 编译默认项目
1. `cmd`运行`git clone --recurse-submodules git@github.com:itgz123/BetaRobot.git my_project`克隆仓库
2. 复制`user_cfg.h.example`为`user_cfg.h`
3. `vscode`中按`Ctrl`+`Shift`+`B`编译默认项目

### 开发自己的`app`
1. 运行`git submodule add <子仓库的URL> [子模块存放的本地路径]`添加自己的子模块
2. 提交到自己的子模块仓库

<子仓库的URL>：这是你想引入的另一个Git仓库的地址，可以是HTTPS或SSH协议。

[子模块存放的本地路径]：这是可选的。如果不指定，Git会自动创建一个与仓库同名的文件夹来存放子模块。

举个例子：假设你想在主项目中添加一个叫 "DbConnector" 的库，可以这样操作：

```bash
git submodule add https://github.com/chaconinc/DbConnector
```
执行后，Git会自动克隆这个仓库，并创建一个名为 DbConnector 的文件夹。
