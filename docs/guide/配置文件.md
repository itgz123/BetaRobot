# 配置文件

## 配置选项

1. 开发板
2. CubeMX 配置
3. APP
4. 其他选项

## 配置文件

| 文件 | 位置 | 职责 |
|------|------|------|
| `app_cfg.h` | `app/Inc/` | 开发板选择、应用参数配置 |
| `user_cfg.h` | 项目根目录 | 用户自定义配置 |
| `bsp_cfg.h` | `bsp/Inc/` | 板载资源枚举、硬件映射 |
| CMakeLists.txt | 项目根目录 | 编译配置 |

## 作用

1. 被其他源文件包含（`#include`）
2. 被 CMake 检测

---

## 配置传递机制

```
┌─────────────────────────────────────────────────────────────┐
│  app_cfg.h（唯一修改点）                                     │
│  - DEVELOPMENT_BOARD 宏选择开发板                            │
│  - 应用参数（任务频率、底盘参数等）                          │
└─────────────────────────────────────────────────────────────┘
                          ↓ 包含
┌─────────────────────────────────────────────────────────────┐
│  bsp_cfg.h（自动适配）                                       │
│  - 根据 DEVELOPMENT_BOARD 自动配置实例数量                   │
│  - 定义板载资源枚举（GPIO/TIM/USART）                        │
│  - 声明硬件映射数组                                          │
└─────────────────────────────────────────────────────────────┘
```

## 切换开发板

```c
// 只需修改 app_cfg.h 中的一个宏
#define STM32F407VET6 0
#define DM_MC02 1
#define DJI_A 2
#define DJI_C 3

#define DEVELOPMENT_BOARD STM32F407VET6  // ⬅️ 修改此处
```
