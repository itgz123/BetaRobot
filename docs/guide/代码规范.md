# 代码规范

本文档总结了 test_my_frame 项目的统一代码规范，适用于 BSP、DRV、APP 三层。

---

## 一、命名规范

### 1.1 快速参考表

| 类别 | 规范 | 示例 |
|------|------|------|
| **变量** | snake_case（下划线分隔，全小写） | `chassis_vx`, `motor_lf` |
| **函数** | PascalCase（大驼峰）+ 模块前缀 | `DJIMotorInit()`, `ChassisTask()` |
| **复杂结构体** | `_s` 后缀 | `Motor_Controller_s`, `Chassis_Ctrl_Cmd_s` |
| **简单数据类型** | `_t` 后缀 | `attitude_t`, `DWT_Time_t` |
| **枚举类型** | `_e` 后缀 | `chassis_mode_e`, `Motor_Type_e` |
| **实例类型** | `Instance` 后缀 | `DJIMotorInstance`, `CANInstance` |
| **宏定义** | UPPER_SNAKE_CASE（全大写下划线分隔） | `DJI_MOTOR_CNT`, `PI` |
| **文件** | lower_case.h/c（全小写下划线分隔） | `dji_motor.c`, `robot_def.h` |

### 1.2 指针变量

- 指针变量命名通常**不带 `p` 前缀**
- 成员变量中的指针使用 `_ptr` 后缀

```c
static DJIMotorInstance *yaw_motor;
static attitude_t *gimba_IMU_data;
```

### 1.3 函数命名

```c
// 普通函数：大驼峰 + 模块前缀
void RobotInit(void);
DJIMotorInstance *DJIMotorInit(Motor_Init_Config_s *config);

// 回调函数：Callback 后缀
static void DJIMotorLostCallback(void *motor_ptr);

// 初始化函数：Init 后缀
DJIMotorInstance *DJIMotorInit(Motor_Init_Config_s *config);
```

### 1.4 文件命名

| 层级 | 文件前缀 | 示例 |
|------|----------|------|
| APP | `app_` | `app_robot.h`, `app_chassis.c` |
| BSP | `bsp_` | `bsp_usart.h`, `bsp_gpio.c` |
| DRV | `drv_` | `drv_motor.h`, `drv_pid.c` |

## 二、代码格式规范

- 使用 **4 空格缩进**（不使用 Tab）
- 运算符两侧加空格，逗号后加空格
- 大括号使用 **Allman 风格**（大括号独占一行）
- 每行代码不超过 **100 字符**

```c
// 正确示例
void ExampleFunction(int a, int b)
{
    int result = a + b;
    if (result > 0)
    {
        // do something
    }
}
```

## 三、注释规范

### 文件头注释

```c
/**
 * @file file_name.c
 * @brief 文件简要描述
 * @author 作者
 * @date 创建日期
 */
```

### 函数注释

```c
/**
 * @brief 简要描述函数功能
 * @param param1 参数1说明
 * @param param2 参数2说明
 * @return 返回值说明
 * @note 注意事项
 */
```

### 代码注释

- 注释应解释 **为什么** 而非 **是什么**
- 使用英文注释（与代码风格一致）

## 四、单位规范

**所有变量、宏定义使用国际单位制（SI）**。

| 物理量 | 单位 | 示例 |
|--------|------|------|
| 长度 | 米（m） | `chassis_width = 0.4f` |
| 速度 | 米/秒（m/s） | `max_speed = 3.0f` |
| 角度 | 弧度（rad） | `yaw_angle = 1.57f` |
| 角速度 | 弧度/秒（rad/s） | `angular_speed = 3.14f` |
| 时间 | 秒（s） | `dt = 0.001f` |
| 电流 | 安培（A） | `motor_current = 5.0f` |
