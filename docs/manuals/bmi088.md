# BMI088 六轴惯性测量单元

BMI088 是一款高性能 6 轴惯性测量单元（IMU），集成了 3 轴加速度计和 3 轴陀螺仪。

## 接口

- 通信协议：SPI
- 加速度计量程：±3g / ±6g / ±12g / ±24g
- 陀螺仪量程：±125°/s / ±250°/s / ±500°/s / ±1000°/s / ±2000°/s
