# DJI C 型开发板

RoboMaster 开发板 C 型，基于 STM32F407 微控制器，集成 BMI088 和 IST8310。

## 规格

- 主控：STM32F407xx
- 板载 BMI088 六轴 IMU
- 板载 IST8310 磁力计
