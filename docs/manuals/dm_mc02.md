# DM-MC02 开发板

达妙科技 DM-MC-Board02 电机开发板，基于 STM32H723 微控制器。

## 规格

- 主控：STM32H723xx（Cortex-M7）
- 强大的 DSP 和浮点运算能力
- 支持 CORDIC 硬件加速
- 适用于高性能电机控制
